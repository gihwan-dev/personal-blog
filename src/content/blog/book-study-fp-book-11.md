## 함수를 리턴하는 함수

예제

```js
function wrapLogging(f) {
  try {
    f();
  } catch (e) {
    sendLogToSomeWhere(e);
  }
}
```

이런식으로 함수를 래핑하는 함수(팩토리)를 생성해 재사용할 수 있다.
